# BiometricPushServer

ASP.NET Core biometric attendance server with provider/client dashboard support.

## Highlights

- Provider dashboard with client selection flow
- Client-scoped dashboard and attendance reporting (Daily/Weekly/Monthly)
- First In / Last Out attendance summaries
- Excel report export (`.xlsx`)
- Client-timezone-aware report aggregation and display

See `/docs/HOW_TO_USE.md` for setup and API usage.
